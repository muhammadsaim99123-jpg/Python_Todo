# Python_Todo
Python Todo App A lightweight, feature-rich todo list application that runs entirely in the command line. Perfect for developers who want to manage tasks without leaving the terminal.
 Features
Add tasks with optional due dates and priorities

List tasks with clear formatting and visual indicators

Mark tasks as complete with status tracking

Delete or clear tasks individually or all at once

Persistent storage – tasks saved locally in JSON format

Priority levels (High, Medium, Low)

Filter tasks by status (pending/completed) or priority

Search functionality to find tasks by keywords

Data export to CSV or text files
